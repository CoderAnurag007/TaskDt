
# API Documentation

## Installation

Before running the application, please ensure that you have the following prerequisites installed on your system:

- [Node.js](https://nodejs.org) (version 16 or later)
- [MongoDB](https://www.mongodb.com) 

Once you have installed Node.js and MongoDB, follow the steps below to set up the application:

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Run the following command to install the dependencies:
